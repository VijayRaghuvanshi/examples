1. ArcheryGameBootstrap is the main class for the application.
2. In order to execute the application execute ArcheryGameBootstrap class.
3. ArcheryGameBootstrap.getPlayersAttempt() method prepares input for the application.
4. As more focus was on to create a generic loosely coupled sport-tournament solution wiht high focus on opps concept, generics, design patterns. So because of time constraint could not write junits for the application.
5. Did a thorough requirement analysis and tried to create an application which is able to support not just an archery but also easily enhanced for other sports as well.
6. Player and Round factory creates metadata for players and a round for archery with the assumption that in real application it can be integrated to persistent store to fecth the meta data.
7. It is a simple java application minimum jdk8 is required.
